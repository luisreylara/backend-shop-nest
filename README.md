# Teslo API

1. Clonar proyecto
2. `npm install`
```
npm install
```
   
4. Clonar el archivo `.env.template` y renombrarlo a `.env`
5. Cambiar las variables de entorno
6. Levantar la base de datos

```
docker compose up -d
```

6. Levantar:

```
npm run start:dev

```

7. Ejecutar SEED

```
http://localhost:3000/api/seed
```
